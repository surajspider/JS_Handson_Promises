Q1. Write one example explaining how you can write a callback function.

Answer:

        function do_homework(callb) {
            setTimeout(() => {
                console.log("do homework");
                callb();
            }, 1000);
        
        }
        function copy_hw() {
            console.log("copy hw");
        }
        do_homework(copy_hw);

Output:

do homework, copy hw

Reason: The function which is passed as an argument inside of another function is called callback function. Here, do_homework() function is called with an argument as another function copy_hw(). Then, it is copied to a parameter in the do_homework() as callb. After 1 sec, setTimeout() starts executing and prints do homework first. Then, callb() is called, and then copy hw is printed!

Q4. Create a promise function accepting an argument, if yes is passed to the function then it should go to resolved state and print Promise Resolved, and if nothing is passed then it should go to reject the state and catch the error and print Promise Rejected.

Answer:

        function demo(value) {
            return new Promise((resolve, reject) => {
                if (value == "yes") {
                    resolve("Promise Resolved");
                }
                else {
                    reject("Promise Rejected");
                }
            })
        }
        demo("yes")
            .then((result) => {
                console.log(result);
            });
            .catch((result) => {
                console.log(result);
            });

Output:

Promise Resolved
